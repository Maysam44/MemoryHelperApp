# تطبيق رفيق الذاكرة (MemoryHelperApp) 🧠❤️

تطبيق مخصص لمساعدة مرضى الزهايمر وعائلاتهم من خلال تنظيم الذكريات، الأدوية، والتواصل الذكي.

## 🛠️ التحديثات الأخيرة والإصلاحات (مارس 2026)

تم إجراء مراجعة شاملة للكود وإصلاح المشاكل التقنية لضمان تجربة مستخدم مثالية:

### 1. توحيد مسارات البيانات (Firebase Path Sync)
- تم حل مشكلة "تضارب المسارات" حيث كان مقدم الرعاية يحفظ في مكان والمريض يقرأ من مكان آخر.
- المسار الموحد لبنك الذكريات الآن هو: `users/{uid}/memoryBank`.
- تم توحيد أسماء الحقول (`relation`, `imageUri`) عبر جميع الشاشات (إضافة، تعديل، عرض).

### 2. تفعيل الذكاء الاصطناعي (Gemini AI)
- تم إصلاح ملف `ai-chat.tsx` ليعمل بشكل آمن وديناميكي.
- يتم الآن جلب مفتاح API من إعدادات Expo (`expo-constants`) بدلاً من القيم النصية الثابتة.
- تحسين الردود التلقائية (Fallback) لتعمل حتى في حال وجود مشاكل في الاتصال.

### 3. إنشاء الشاشات المفقودة
تم إنشاء الشاشات التالية بالكامل لتمكين مقدم الرعاية من إدارة التطبيق:
- **إضافة دواء (`add-medicine.tsx`):** لإضافة الأدوية وجدولة التنبيهات.
- **إضافة منطقة آمنة (`add-safe-zone.tsx`):** لتحديد نطاق جغرافي آمن للمريض.
- **إرسال رسالة صوتية (`add-voice-note.tsx`):** لإرسال رسائل صوتية حنونة تظهر في واجهة المريض.

### 4. تحسين واجهة المريض (Patient UI)
- تبسيط واجهة "بنك الذكريات" للمريض لتكون أكثر سهولة (عرض فقط مع إمكانية سماع الصوت).
- إزالة أزرار التعديل والحذف من واجهة المريض لتقليل التشتت.
- تحسين استقرار تشغيل الملفات الصوتية.

## 🚀 كيفية التشغيل
1. تأكد من وجود ملف `.env` يحتوي على مفاتيح Firebase و Gemini.
2. قم بتثبيت المكتبات: `npm install` أو `pnpm install`.
3. ابدأ المشروع: `npx expo start`.
